                                                 MiPatcher


    MiPatcher was made in Windows but is intended to be used in a Linux box that has WINE, after CyanogenMod has finished building.


With that being said, If you have a base that meets the required conditions then obviously this can be used in Windows as it does not depend on any Linux functionality.



Requirements
- Built CyanogenMod base with init.d support and NO SELinux.
- WinRAR or 7Zip (To make a flashable zip)
   Download: MiPatcher-KK-0.03


How-to
1 ) Extract your CyanogenMod contents into ‘Base'

Make sure the following files have proper MiCode, included in files of the same name located in 'Donor' folder as a reference

- Build.prop

    // MIUIv5 Code Flags

- Updater-script

    // Shelld and LBEsec code flags

- Framework-res.apk

    // Storage-list & power-profile.xml

2 ) Check 'DRM Patch' to avoid making manual kernel edits

3 ) Select your device resolution and check 'Set new DPI’.

4 ) Hit 'Apply Patches’

5 ) Zip the contents of 'Base' when patching is done, then flash it.

F.A.Q
Q: What do I need to run this in Linux?
A: WINE, which can be found in the software center usually :>

Q: All I need is a CyanogenMod 11 base?
A: Yes, as long as it does not have SELinux you can apply MIUI to it with this tool.

Q: I wiped Donor folder!
A: Donor contains a the core files needed for MIUI v5 to work.. extract them from the zip file again they are system essential.
