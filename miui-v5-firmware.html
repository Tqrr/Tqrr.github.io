                                            MIUI v5 firmware


   The following builds of MIUI v5 are from the RTM 4.11.xx branch, they are based on the final revision of CM11 code: KTU84Q

Every build here can be produced using MiPatcher + Source building Cyanogen without SELinux. In any case, as I have time I will continue to upload builds.


// RTM 1 //

- NFC Force-quit may occur (Laziness)

- Resources may not be properly aligned in some devices

- Incomplete 'Stock' APK list (No Market, GameCenter etc.. some would consider this a 'feature' but for completion sake..

- No Native Wifi tether

- Potentially stable in most devices, xHDPI only

// RTM 2 //

- NFC errors should be resolved

- Proper resources

- No Native Wifi tether

- Still garbage in text

- Stock MIUI APKs (sans camera)

- Assumed stable in all devices, HDPI xHDPI xxHDPI supported, xxHDPI & HDPI support will be added to multibases as seperate branches.
